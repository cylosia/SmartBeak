
# Failure Drills

## Goal
Practice failure so real incidents are boring.

---

## Publishing Drill (Monthly)
1. Disable a publish target
2. Observe job failures
3. Verify DLQ population
4. Retry jobs
5. Confirm recovery

## Region Failure Drill (Quarterly)
1. Simulate region outage
2. Ensure other regions unaffected
3. Verify jobs queue correctly
4. Restore region

---

Document findings after each drill.
