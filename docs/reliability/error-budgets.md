
# Error Budgets

## Philosophy
Error budgets are used to **protect velocity**.
When a budget is exhausted, feature work pauses.

---

## Budgets

### Content Creation
- Error budget: 0.1%
- Burn rate alert: 2x over 1 hour

### Publishing
- Error budget: 0.5%
- Burn rate alert: 2x over 6 hours

---

## Actions on Exhaustion
- Freeze deployments
- Investigate root cause
- Apply mitigations before resuming work
