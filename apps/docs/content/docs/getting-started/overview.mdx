This section introduces the essentials of using SmartBeak.

You will learn how organizations, domains, content, publishing, and analytics fit together so you can move from setup to production quickly.

Before continuing, make sure you can:

- Access the correct organization or workspace
- Add or manage the domains you want to operate on
- Open the SmartBeak content, publishing, and analytics areas
